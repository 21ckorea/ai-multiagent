================================================================================
파일명: gemini-image-style.txt
용도  : 블로그 대표 이미지(썸네일) 생성 스타일 프롬프트 접미사(suffix)
목적  : Gemini 이미지 생성 시, 각 섹션의 imgplace 설명 텍스트 뒤에 이 내용을
        덧붙여 이미지의 통일된 스타일(3D 입체 일러스트, 파스텔톤 등)을 지정.
        gemini_image.js 의 GEMINI_IMAGE_PROMPT_SUFFIX 상수 대신 이 파일을 읽어 사용.
사용처: automation/gemini_image.js → buildGeminiImagePrompt(imageInnerPrompt)
주의 : 이 파일의 내용은 이미지 주제 설명(imgplace.text) 뒤에 자동으로 합쳐짐.
       "앞서 제시한 주제를..."로 시작하므로 단독 사용 시 주제 설명이 앞에 와야 함.
================================================================================

 앞서 제시한 주제를 직관적으로 표현하는 고품질 블로그 썸네일 이미지를 생성해줘. 이미지의 가로세로 비율은 반드시 16:9 비율(가로가 긴 와이드스크린 형태)로 생성해줘. 스타일은 아주 세련되고 트렌디한 3D 입체 일러스트(Isometric 3D) 형태로 그려줘. 화사하고 밝은 파스텔톤 배경 위에 주제와 관련된 멋진 오브젝트나 아이콘들이 세련되게 배치된 모습이어야 해. 부드러운 조명과 세련된 질감을 표현해서 고급스럽고 눈에 띄게 만들어줘. 복잡하지 않고 깔끔하며 미니멀한 구성으로, 글자나 텍스트는 절대 포함하지 말아줘. 인물이나 특정 직책(예: 대통령 등)을 묘사할 때는 과거 인물이 아닌 2026년 현재 대한민국의 최신 현실을 반드시 반영해서 그려줘(예: 현재 대한민국 대통령은 이재명).
